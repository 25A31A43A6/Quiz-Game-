Quiz Game: The Quiz Game Project is a console-based application developed using C++ programming language. 
This project is designed for beginners to understand basic programming concepts such as variables, conditions, loops, and user input/output operations.
In this application, users are asked multiple-choice questions one by one. 
The program checks whether the selected answer is correct or incorrect and updates the score accordingly.
The project is simple, interactive, and easy to understand, making it a good mini project for students and beginners learning C++ programming.
